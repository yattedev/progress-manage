# game
## Installation Redis  
### MacOSX
```$ brew install redis ```
### Linux
```$ wget http://download.redis.io/releases/redis-4.0.10.tar.gz ```  
```$ tar zxvf redis-4.0.10.tar.gz ```  
```$ cd redis-4.0.10 ```  
```$ make ```  
```$ sudo make install ```  

## Installation Redis Client for node.js
```$ npm install redis ```  
or  
```$ yarn redis ```  

## Installation jest  
``` $ npm install jest ```  
or  
``` $ yarn jest ```

## Getting Started    
starting redis-server  
```$ redis-server ```

## Testing 
after starting redis-server  
```$ npm run test```
